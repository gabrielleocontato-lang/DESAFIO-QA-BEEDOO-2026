# DESAFIO-QA-BEEDOO-2026

Qual você acredita ser o objetivo da aplicação?
O objetivo é cadastrar cursos e divulgar uma lista de fácil visualização.

Quais são os principais fluxos disponíveis
Cadastro de curso - nome, descrição, imagem, foto de capa, data de inicio e fim, vagas, modalidade e link e endereço.
Visualização dos cursos - card com foto, nome e descrição, data e vagas

Quais pontos do sistema você considera mais críticos para teste?
1 - Fluxo de exclusão de curso 2 - Edição de cursos após criação 3 - Campo de número de vagas 4 - Limite de caracteres para nome do curso e descrição, com o objetivo de manter a exibição correta dos cards
